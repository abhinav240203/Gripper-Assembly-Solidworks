# Servo-Driven Robotic Gripper

## Overview

This project is a servo-actuated two-finger robotic gripper designed in SolidWorks. The mechanism uses a pair of meshing gears and linkage arms to generate synchronized jaw movement for gripping and releasing objects. The design is compact, simple, and suitable for robotic manipulation and automation applications.

## Features

* Single servo motor actuation
* Synchronized dual-jaw movement
* Gear-linkage transmission mechanism
* Compact and lightweight design
* Suitable for robotic arm integration
* Easy to manufacture and assemble

## Working Principle

The servo motor drives the primary gear, which meshes with a secondary gear. As the gears rotate, the attached linkage arms transfer motion to the gripper jaws. 
This causes both jaws to move simultaneously, allowing the gripper to grasp and release objects efficiently.
